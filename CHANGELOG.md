# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-28

### Added
- Initial release
- MorphingNavigationScaffold - main widget for easy integration
- MorphingNavigationTheme - customizable theme configuration
- MorphingNavHeader - configurable sidebar header
- MorphingNavFooter - configurable sidebar footer
- NavItem model with support for nested items (sections)
- Smooth morphing animation between sidebar and tab bar modes
- Responsive breakpoints for automatic mode switching
- Glassmorphism effect in tab bar mode
- Staggered expand/collapse animations for sections
- Animated dropdown for sections in tab bar mode
- Keyboard shortcut (T) to toggle modes
- Badge support for navigation items
- Light and dark theme presets
